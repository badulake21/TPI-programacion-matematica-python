# Gestion de Datos de Paises en Python

Este proyecto es una aplicacion de consola desarrollada en Python para gestionar informacion de paises usando listas, diccionarios, funciones y archivos CSV.

## Funcionalidades

- Leer paises desde un archivo CSV.
- Mostrar todos los paises cargados.
- Agregar un nuevo pais.
- Actualizar poblacion y superficie de un pais.
- Buscar paises por nombre.
- Filtrar por continente.
- Filtrar por rango de poblacion.
- Filtrar por rango de superficie.
- Ordenar por nombre, poblacion o superficie.
- Mostrar estadisticas basicas.

## Como ejecutar el programa

1. Abrir una terminal en la carpeta del proyecto.
2. Ejecutar:

```bash
python main.py
```

## Formato del CSV

El archivo `paises.csv` debe tener estas columnas:

```csv
nombre,poblacion,superficie,continente
Argentina,45376763,2780400,America
Japon,125800000,377975,Asia
```

